    **`_safe_filename(name: str) -> str`** (private):
    Sanitize filesystem-invalid chars: `re.sub(r'[/\\:*?"<>|]', '-', name).strip()`. Used for both People and Source note filenames.

    **`_normalize_person(name: str) -> str`** (private):
    `name.strip().title()` — prevents duplicate notes from casing differences per RESEARCH.md Pitfall 2.

    **`_append_backlink_to_note(note_path: Path, note_title: str, backlink_line: str, slug: str) -> None`** (private):
    Core append-only logic used by both People and Source notes:
    1. If file does not exist: create with `f"# {note_title}\n\n## Articles\n\n"`, `mkdir(parents=True, exist_ok=True)` on parent.
    2. Read file content.
    3. Find `"## Articles"` marker. If missing, append `"\n\n## Articles\n\n"` to content.
    4. Idempotency: check if `f"[[{slug}|"` is present in content after the `## Articles` marker. If yes, return immediately.
    5. Append `backlink_line` at end of file: `content.rstrip("\n") + f"\n{backlink_line}\n"`.
    6. Write back with `write_text(updated, encoding="utf-8")`.

    **`update_people_notes(record: ArticleRecord, vault_path: str) -> None`** (public):
    For each person in `record.person`:
    - Normalize: `_normalize_person(person)`
    - Path: `Path(vault_path) / "People" / f"{_safe_filename(normalized)}.md"`
    - Title: normalized person name
    - Date string: `record.article_data.publish_date.isoformat() if record.article_data.publish_date else "undated"`
    - Backlink: `f"- [[{record.slug}|{record.article_data.title}]] ({record.source_name}, {date_str})"`
    - Call `_append_backlink_to_note(path, title, backlink, record.slug)`
    If `record.person` is empty, return immediately (no guessing per CONTEXT.md decision).

    **`update_source_note(record: ArticleRecord, vault_path: str) -> None`** (public):
    - Path: `Path(vault_path) / "Sources" / f"{_safe_filename(record.source_name)}.md"`
    - Title: `record.source_name` (original, unsanitized, for the `# H1` heading)
    - Date string: same as above
    - Backlink: `f"- [[{record.slug}|{record.article_data.title}]] ({date_str})"`
    - Call `_append_backlink_to_note(path, title, backlink, record.slug)`

    **`regenerate_index(vault_path: str) -> None`** (public):
    1. `articles_dir = Path(vault_path) / "Articles"`
    2. Glob `articles_dir.glob("*/metadata.json")`, read each with `json.loads()`, skip on error.
    3. Sort all records by `r.get("date") or "0000-00-00"` descending.
    4. Group by person using `collections.defaultdict(list)`. Articles with empty `person` list go to key `"__unlinked__"`.
    5. Build output lines:
       - `"# Article Index"`, blank line, `f"_Last updated: {date_cls.today().isoformat()}_"`, blank line
       - For each person (sorted alphabetically, excluding `__unlinked__`):
         - `f"## {person} ({len(articles)} articles)"`
         - For each article (re-sorted reverse-chronological):
           - `f"- [[{slug}|{title}]] ({source}, {date})"`
       - If `__unlinked__` has entries:
         - `f"## Unlinked Articles ({len(unlinked)} articles)"`
         - Same article format
    6. Write to `articles_dir / "_index.md"` with `write_text("\n".join(lines), encoding="utf-8")`.
    7. If `articles_dir` does not exist or has no metadata.json files, write a minimal index: `"# Article Index\n\n_No articles archived yet._\n"`.

    **`update_graph(record: ArticleRecord, config: AppConfig) -> None`** (public):
    Single entry point called from archiver.py. Calls in sequence:
    1. `update_people_notes(record, config.vault.path)`
    2. `update_source_note(record, config.vault.path)`
    3. `regenerate_index(config.vault.path)`
    Each call wrapped in its own `try/except Exception` — log error via `logger.error(...)` but never raise. Graph failures are non-fatal per RESEARCH.md anti-patterns.

    **Logger:** `logger = logging.getLogger(__name__)`
