Create `src/mouse_research/searcher.py` with:

1. **SearchResult dataclass** (at top of file):
```python
@dataclass
class SearchResult:
    number: int          # 1-based display index (assigned after dedup filtering)
    title: str           # Newspaper name
    date: str            # ISO date "YYYY-MM-DD"
    location: str        # Publication location
    url: str             # Full Newspapers.com URL
    keyword_matches: int # Match count
```

2. **LOCATION_CODES dict** mapping lowercase state names to region codes:
```python
LOCATION_CODES = {
    "pennsylvania": "us-pa",
    "new york": "us-ny",
    "maryland": "us-md",
    "new jersey": "us-nj",
    "delaware": "us-de",
    "virginia": "us-va",
    "west virginia": "us-wv",
    "ohio": "us-oh",
}
```

3. **resolve_location(location: str) -> str**: Lowercase strip, lookup in LOCATION_CODES, return match or pass through unchanged.

4. **ScraperError(Exception)**: Custom exception for scraper failures.

5. **call_scraper(keyword: str, years: str | None = None, location: str | None = None, max_pages: int | None = None) -> list[dict]**:
   - Build command: `[shutil.which("node"), str(SCRAPER_WRAPPER), "--keyword", keyword]`
   - Where `SCRAPER_WRAPPER = Path.home() / ".mouse-research" / "scraper-wrapper.js"`
   - If years: validate format with `re.fullmatch(r'\d{4}(-\d{4})?', years)`, raise ValueError if invalid, append `["--years", years]`
   - If location: resolve via `resolve_location(location)`, append `["--location", resolved]`
   - If max_pages: append `["--max-pages", str(max_pages)]`
   - Call `subprocess.run(cmd, capture_output=True, text=True, timeout=300)`
   - If returncode != 0: check stderr for "Cloudflare" — if found, raise ScraperError with message including "Session may have expired -- run `mouse-research login newspapers.com`". Otherwise raise ScraperError with stderr content.
   - Parse stdout with `_parse_scraper_output(result.stdout)`
   - Return list of dicts

6. **_parse_scraper_output(stdout: str) -> list[dict]**: Split by newlines, json.loads each non-empty line, skip JSONDecodeError lines silently, return list.

7. **search_and_filter(keyword, years, location, config: AppConfig, max_pages: int | None = None) -> tuple[list[SearchResult], int]**:
   - Call `call_scraper(keyword, years, location, max_pages)`
   - Filter: for each raw dict, call `is_duplicate(config.vault.path, raw["url"])` — exclude duplicates
   - Count excluded
   - Build `SearchResult` list from non-duplicates with 1-based numbering, mapping raw dict keys: `title` -> title, `date` -> date, `location` -> location, `url` -> url, `keywordMatches` -> keyword_matches
   - Return `(results, excluded_count)`

8. **parse_selection(selection: str, max_count: int) -> list[int]**:
   - Strip and lowercase input
   - If "all": return list(range(max_count))
   - Split by comma, for each part:
     - If contains "-": split on "-" (maxsplit=1), parse start/end as int, convert to 0-based (subtract 1), validate bounds (0 <= start, end < max_count, start <= end), extend indices with range(start, end+1)
     - Else: parse as int, convert to 0-based, validate 0 <= idx < max_count, append
   - On any int() failure or out-of-bounds: raise ValueError with descriptive message
   - Return sorted(set(indices))
