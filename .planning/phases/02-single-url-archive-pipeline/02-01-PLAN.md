**Module structure:**

```python
"""Playwright-based page fetcher for mouse-research pipeline.

Handles two fetch paths:
  1. Newspapers.com: network interception for full-page JPG, DOM crop, 500px resize
  2. Modern web articles: screenshot + HTML only

Critical constraints (Phase 1 validated):
  - channel="chrome" required on macOS — Playwright Chromium fails with ValidationError 54
  - GLM-OCR crashes on images >~500px — article_image_path MUST be ≤500px max dim
  - GLM-OCR hallucinates on full-page images — MUST crop to article region first
"""
```

**Public interface:**

```python
def fetch_url(url: str, config: AppConfig, output_dir: Path) -> FetchResult:
    """Fetch a page and return raw artifacts.

    For Newspapers.com URLs: also extracts and crops the page image.
    For all URLs: captures screenshot and HTML.

    Args:
        url: The article URL to fetch
        config: AppConfig with browser settings and OCR settings
        output_dir: Directory to save screenshot and image files (must exist)

    Returns:
        FetchResult with all captured artifacts
    """
```

**Implementation requirements:**

1. Detect Newspapers.com: `"newspapers.com" in url.lower()`

2. Load cookies via `load_cookies("newspapers.com")` (or relevant domain from URL). Pass to `browser.new_context(storage_state=storage)` if cookies exist.

3. Playwright launch: ALWAYS use `channel="chrome"` on macOS. Never use bundled Chromium without the channel override:
   ```python
   browser = p.chromium.launch(headless=config.browser.headless, channel="chrome")
   ```

4. For Newspapers.com: register response listener BEFORE goto:
   ```python
   page_image_url: list[str] = []  # use list for nonlocal mutation

   def capture_image(response):
       if "img.newspapers.com" in response.url and response.status == 200:
           ct = response.headers.get("content-type", "")
           if "image" in ct and not page_image_url:
               page_image_url.append(response.url)

   page.on("response", capture_image)
   page.goto(url, wait_until="networkidle", timeout=30000)
   ```

5. Screenshot: `page.screenshot(path=str(output_dir / "screenshot.png"), full_page=True)`

6. HTML: `html = page.content()`

7. For Newspapers.com page image download (if intercepted):
   - Download via `httpx.get(page_image_url[0], follow_redirects=True, timeout=30)` with cookies from storage state if available
   - Save as `output_dir / "page_image.jpg"`
   - If interception failed, attempt URL construction fallback: parse image ID from URL path (`/image/(\d+)`), construct `https://img.newspapers.com/img/img?id={image_id}&width=2000`

8. Article crop strategy (implement both, A with B as fallback):

   **Option A — DOM bounding box:**
   ```python
   bbox = page.evaluate("""() => {
       const selectors = [
           '.clip-region', '[class*="clip"]', '[class*="article-image"]',
           '[class*="ArticleImage"]', '[data-testid*="article"]'
       ];
       for (const sel of selectors) {
           const el = document.querySelector(sel);
           if (el) {
               const r = el.getBoundingClientRect();
               if (r.width > 50 && r.height > 50) {
                   return {x: r.x, y: r.y, width: r.width, height: r.height,
                           selector: sel};
               }
           }
       }
       return null;
   }""")
   ```
   
   If bbox is not None: use Pillow to crop the downloaded page_image.jpg using the bbox coordinates. The DOM coordinates are in CSS pixels; the page image is the full newspaper scan at its native resolution — use the page's scroll dimensions to compute the scale factor:
   ```python
   page_dims = page.evaluate("() => ({w: document.scrollingElement.scrollWidth, h: document.scrollingElement.scrollHeight})")
   # scale = page_image.width / page_dims["w"]
   # crop_box = (bbox["x"]*scale, bbox["y"]*scale, (bbox["x"]+bbox["width"])*scale, (bbox["y"]+bbox["height"])*scale)
   ```

   **Option B — Center-crop fallback (when DOM extraction returns None or crop is <100x100px):**
   ```python
   # Crop center 80% width × 60% height of the full page image
   w, h = img.size
   left = int(w * 0.10)
   top = int(h * 0.20)
   right = w - left
   bottom = top + int(h * 0.60)
   cropped = img.crop((left, top, right, bottom))
   ```

9. Resize cropped article image to 500px max dimension using Pillow:
   ```python
   w, h = cropped.size
   scale = 500 / max(w, h)
   if scale < 1.0:
       cropped = cropped.resize((int(w*scale), int(h*scale)), Image.Resampling.LANCZOS)
   cropped.save(str(output_dir / "article_image.jpg"), "JPEG", quality=85)
   ```

10. Return `FetchResult(url=url, html=html, screenshot_path=output_dir/"screenshot.png", page_image_path=..., article_image_path=..., is_newspapers_com=True/False)`

11. Wrap entire Playwright block in try/except. On failure: re-raise as `FetchError(url, reason)` — define a simple exception class in this module.

**Helper to extract domain from URL for cookie loading:**

from urllib.parse import urlparse
